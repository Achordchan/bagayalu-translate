# 隐私条款 / Privacy Policy

更新时间：2025-12-22

《大佐翻译官》（以下简称“本应用”）重视你的隐私。本文件说明本应用在 macOS 上的权限使用、数据处理方式以及第三方服务调用范围。

---

## 1. 我们收集哪些数据？

本应用 **不主动收集** 你的个人身份信息（如姓名、手机号、账号信息）。

本应用可能会在本地处理或临时使用以下数据：

- 你输入的待翻译文本
- 你通过截图框选得到的图像（用于 OCR）
- OCR 识别出的文字
- 你复制到剪贴板的内容（用于“读取/写入剪贴板”功能）
- 应用内部的运行日志（用于排查错误，通常保存在本机）

---

## 2. 权限使用说明

### 2.1 屏幕录制（Screen Recording）
- **用途**：用于截图框选区域，获取图像以便 OCR。
- **范围**：仅在你触发截图功能并完成框选时，读取对应区域画面。

### 2.2 辅助功能（Accessibility）
- **用途**：用于全局快捷键监听、部分窗口行为控制。
- **范围**：仅用于应用功能所需的事件监听/窗口行为，不会记录你的键盘内容用于其他目的。

### 2.3 剪贴板（Clipboard）
- **用途**：
  - 将 OCR 识别出的文本复制到剪贴板
  - 将截图图片复制到剪贴板
- **范围**：仅在你点击“复制/提取”等按钮时写入；在启用“监控双击复制”时会读取剪贴板变化用于触发翻译。

---

## 3. OCR 数据如何处理？

- OCR 识别使用 macOS 本地的系统能力（Apple Vision），**默认在本机完成识别**。
- OCR 图像与识别文本不会被我们上传到自建服务器。

---

## 4. 翻译请求与第三方服务

本应用支持多种翻译引擎。根据你的选择，翻译内容可能会发送到第三方服务：

- **Google 翻译**：会将待翻译文本发送到 Google 翻译相关接口。
- **OpenAI Compatible**：会将待翻译文本发送到你配置的 BaseURL 所对应的服务（可能是 OpenAI 或兼容 OpenAI 协议的第三方）。

注意：

- 发送的内容通常包含：待翻译文本、目标语言等必要参数。
- 我们不对第三方服务的隐私策略负责。请你在使用前阅读对应服务商的隐私政策。

---

## 5. API Key 如何保存？

当你使用 OpenAI Compatible 引擎时：

- API Key 会保存在 macOS **Keychain（钥匙串）**中。
- 本应用不会将你的 API Key 上传到我们自建服务器。

---

## 6. 数据存储与删除

- 主要数据（设置、缓存、日志）存储在你的本机。
- 你可以通过卸载应用来移除本应用在本机产生的数据（Keychain 项可能需要你在钥匙串中手动删除）。

---
